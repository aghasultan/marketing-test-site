To spin up the project locally:

1.  **Clone:** `git clone https://github.com/aghasultan/marketing-test-site.git`
2.  **Install:** `npm install`
3.  **Run:** `npm run dev`
4.  **Edit:** `src/pages/Home.tsx` to modify the landing page.
