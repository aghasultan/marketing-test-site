# Project Overview

## Executive Summary
**Riffat Labs Marketing Site** is a high-performance, interactive web application built to showcase marketing services. It features a sophisticated "Results Engine" for displaying case studies and a multi-step "Apply Wizard" for client onboarding.

## Technical Architecture
- **Type**: Monolithic Single Page Application (SPA)
- **Frontend**: React 18, TypeScript, Vite
- **Styling**: TailwindCSS, Shadcn/UI, Framer Motion
- **Hosting**: Vercel

## Key Features
1.  **Results Engine**: A dynamic portfolio filtering system allowing users to explore case studies by industry and spend.
2.  **Apply Wizard**: A complex, multi-step form with validation state management for qualifying leads.
3.  **Audit Tool**: An interactive SEO/Marketing audit simulation.
